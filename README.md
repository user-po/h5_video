[![vJaj81.md.png](https://s1.ax1x.com/2022/08/12/vJaj81.md.png)](https://imgtu.com/i/vJaj81)
## 使用方法
```javascript
import video from "xxx";
  video({
          url,
          elem,
          autoplay: true,
      });
```
## 参数
|  参数   | 参数意义  |是否必须|
|  ----  | ----  | ---- |
| url  | 视频地址 |是|
| elem  | 插入到哪一个元素后 |是|
| autoplay  | 是否自动播放 |否(默认不自动播放)|
